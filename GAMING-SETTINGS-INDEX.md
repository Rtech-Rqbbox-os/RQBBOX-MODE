## ⚙️ Installation Scripts

### Main Installer (Use This!)
**File:** `windows/scripts/Install-GamingSettings.bat`

**What it does:**
- Installs Windows Gaming Settings integration
- Creates all registry entries
- Configures performance profiles
- Sets up Game DVR
- Enables Game Bar
- Configures controller

**How to use:**
```
1. Right-click the file
2. Select "Run as Administrator"
3. Click Yes on UAC prompt
4. Wait for completion
```

---

### Alternative: PowerShell
**File:** `windows/scripts/Install-RQBBOXGamingSettings.ps1`

**How to use:**
```powershell
# Open PowerShell as Administrator
cd "C:\Path\to\RQBBOX MODE\windows\scripts"
.\Install-RQBBOXGamingSettings.ps1
```

---

### Uninstall
**File:** `windows/scripts/Uninstall-GamingSettings.bat`

**How to use:**
```
1. Right-click the file
2. Select "Run as Administrator"
3. Type YES to confirm
4. Wait for completion
```
