# Games Catalog

RQBBOX Experience OS 1.0 comes with 6 built-in HTML5 games:

## Neon Drift Racing
- **Category:** Racing
- **Controls:** Left/Right arrows to steer
- **Description:** High-speed neon racing through cyberpunk streets
- **Individual API:** /api/games/neon-drift-racing

## Pixel Quest
- **Category:** Adventure
- **Controls:** Arrow keys to move, Collect gems
- **Description:** Retro pixel-art adventure through mystical lands
- **Individual API:** /api/games/pixel-quest

## Star Fighter X
- **Category:** Action
- **Controls:** Arrow keys to move, Space to shoot
- **Description:** Space combat simulator with intense dogfights
- **Individual API:** /api/games/star-fighter-x

## Void Craft Sandbox
- **Category:** Sandbox
- **Controls:** Left-click to place, Right-click to remove
- **Description:** Build and explore in an infinite void world
- **Individual API:** /api/games/void-craft-sandbox

## Retro Zone
- **Category:** Retro
- **Controls:** Arrow keys to move paddle
- **Description:** Classic arcade breakout game
- **Individual API:** /api/games/retro-zone

## Cube Runner 3D
- **Category:** Action
- **Controls:** Arrow keys to dodge obstacles
- **Description:** Endless runner in a 3D cubic world
- **Individual API:** /api/games/cube-runner-3d
