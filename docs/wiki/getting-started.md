# Getting Started

## Windows

1. Double-click Launch RQBBOX MODE.bat
2. Server starts at http://127.0.0.1:19778/
3. Opens dashboard automatically

## Manual Start

```
npm start
# or
node server.js
# Then open http://127.0.0.1:19778/dashboard
```

## First Boot

When you first launch the dashboard, you will see:

1. **Boot Animation** — Console-style loading bar
2. **Profile Selection** — Choose or create a user profile
3. **Dashboard** — Full-screen gaming interface

## Profile Setup

- Enter a username to sign in
- Or create a new account with the Register button
- STAR users (like than0s230) get a gold badge on the home page

## Navigation

Use the sidebar buttons on the left to switch between:
- Home — Featured game, recent games, quick actions
- Chat — Text and voice chat
- Games — Browse and launch 6 built-in games
- Store — Browse games and apps
- Apps — 16 app integrations
- Captures — Screenshots and recordings
- Settings — Configure everything
