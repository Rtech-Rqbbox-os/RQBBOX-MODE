## What is RQBBOX MODE?

**RQBBOX MODE** is a Console Gaming Experience for PC — a full-screen Xbox-like dashboard with controller navigation, Windows Game Mode integration, Quick Resume technology, 6 built-in HTML5 games, 16 app integrations, and a complete API server. It runs on Node.js and transforms any Windows PC into a gaming console.

## What is RQBBOX OS?

**RQBBOX OS** is a portable USB gaming operating system by RhysTech. It runs entirely from a USB drive — no installation required. Plug it into any PC, start the server, and instantly access games, apps, settings, and saves. Powered by the RQBBOX Kernel with a modular microkernel architecture.
